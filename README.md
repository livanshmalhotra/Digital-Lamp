# Frontend Digital Lamp

A futuristic frontend project showcasing a glowing digital lamp with immersive visuals.
Built using React + TypeScript + Vite for fast development and smooth performance.
Tailwind CSS is used for modern, utility-first styling.
Custom animations and glow effects create a sci-fi aesthetic.
Starfield background enhances depth and ambience.
Layered UI with proper z-index handling ensures visual clarity.
Fully responsive layout across screen sizes.
Optimized for fast refresh and developer experience.
Clean component-based architecture for scalability.
Ideal for experimenting with creative UI and animations.
